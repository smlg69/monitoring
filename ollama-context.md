## Контекст проекта для Ollama

Я разрабатываю React приложение для мониторинга IoT систем.

### Технологический стек:
- React 18
- React Router DOM
- CSS Modules / CSS-in-JS
- Fetch API для запросов
- SVG для схем

### Архитектура:
1. **App.jsx** - корневой компонент с маршрутизацией
2. **ApiContext** - управление токенами API
3. **AccessPage** - главная страница СКУД с:
   - Таблицей оборудования
   - SVG мнемосхемой
   - Графиками параметров
